Zusammenfassung Modul 293

LU01

In "LU01b - Umgebung einrichten" wird die Installation und Konfiguration von Webstorm behandelt. Sie müssen ein Jetbrains-Benutzerkonto erstellen, sich anmelden,
Webstorm herunterladen und installieren. Zudem wird empfohlen, ein GitHub-Benutzerkonto einzurichten und dieses in Webstorm zu konfigurieren.
HTML, die Auszeichnungssprache zur Erstellung von Webseiten, besteht aus Elementen, die durch Tags markiert sind und Attribute enthalten können. CSS (Cascading Style Sheets) sorgt für die optische Gestaltung von HTML-Inhalten. In "LU01d - HTML Grundgerüst" wird das Grundgerüst jeder HTML-Datei beschrieben, das aus den Hauptblöcken !DOCTYPE html, html, head und body besteht.

LU02

In der Lektion "LU02a - Inhalt strukturieren" lernen Sie die HTML-Elemente nav, main, header, footer, section und article kennen, die zur sauberen Strukturierung der Inhalte einer Webseite beitragen. Das zuvor behandelte rudimentäre HTML-Gerüst wird um diese Elemente erweitert. header und footer dienen als Kopf- und Fußzeile, nav enthält die Navigation, und der Hauptinhalt wird im main Bereich strukturiert, unterteilt durch article und section. In "LU02b - Cascading Style Sheets (CSS)" wird die Bedeutung von CSS für das Layout und Design von Webseiten erklärt. CSS regelt die Darstellung der HTML-Inhalte und unterstützt die Trennung von Struktur (HTML) und Gestaltung (CSS). Eine CSS-Regel besteht aus einem Selektor und einer Deklaration. Kommentare in CSS (/* ... */) helfen, den Code zu dokumentieren. Block-Elemente wie Absätze und Listen strukturieren den Hauptinhalt, während Inline-Elemente wie Links und hervorgehobene Wörter Textteile differenzieren. HTML-Dokumente sind hierarchisch aufgebaut, wobei Elemente ineinander verschachtelt werden können.

LU03

In "LU03a - Anwendung von Block-Elementen" werden die Eigenschaften und Anwendungsbereiche von Block-Elementen in HTML besprochen. Block-Elemente beginnen normalerweise auf einer neuen Zeile und nehmen die gesamte verfügbare Breite ein. Beispiele für Block-Elemente sind die Überschriften h1 bis h6, die für verschiedene Hierarchieebenen von Überschriften verwendet werden. Das div-Element dient als Container für andere HTML-Elemente, das p-Element für Textabsätze, und das table-Element zur Darstellung von tabellarischen Daten. In "LU03b - Anwendung von Inline-Elementen" wird der Unterschied zu Block-Elementen erklärt: Inline-Elemente starten nicht auf einer neuen Zeile und ihre Breite wird durch den Inhalt bestimmt. Beispiele sind das span-Element, das als Inline-Container für Textabschnitte dient, der a-Tag für Hyperlinks, und der img-Tag für das Einbetten von Bildern. "LU03c - Selektoren für Elemente" behandelt die Verwendung der Attribute id und class in HTML. Das id-Attribut gibt einem Element eine eindeutige ID, die zur gezielten Auswahl und Manipulation verwendet wird. Das class-Attribut ermöglicht die Gruppierung mehrerer Elemente zur gemeinsamen Formatierung durch CSS oder zur Auswahl durch JavaScript.
